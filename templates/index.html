About the Project

Problem Statement:

To assess students' performance in a coding class, educators have to rely on manual analysis techniques due to a few problems in applying data analytic methods directly on computer programs. Computer programs are processed in the form of abstract syntax trees, whereas machine learning or analytical algorithms require a linear vector as an input.

Project Scope:

Convert AST's of a program into an Encoded Vectorised form and apply data analytic and machine learning techniques on it. Scope of the proposed model is limited to Python programs. Not only useful for educators to evaluate students' proficiency in the language but also directs students towards the right solution.

Application Areas:

  • Latent Space Representation or Vectorisation
  • Educational Data-Mining
  • Coding Performance Evaluation

Project Objectives:

  • Constructing a robust recursive autoencoder to transform python programs to vectors.
  • Visualization Dashboard showing students' progress flow in writing a program.
  • Clustering students based on their understanding and detecting outliers.
